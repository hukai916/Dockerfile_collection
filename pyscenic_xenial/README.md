# A pyscenic image that runs on Xenial
## How to use (Docker):
```bash
docker pull hukai916/pyscenic_xenial:0.1
```

## How to use (Singularity):
```
singularity pull docker://hukai916/pyscenic_xenial:0.1
```

## pyscenic version: 0.11.2
